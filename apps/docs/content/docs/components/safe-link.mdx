A component that wraps next/link, it handles external links in the document.

## Usage

```tsx
import { SafeLink } from "next-docs/link";

<SafeLink href="/docs/components">Click Me</SafeLink>;
```

### Contentlayer

You can pass it as a component in `<MDX>`.

```tsx
//in a server component
import { SafeLink } from "next-docs/link";

const MDX = getMDXComponent(code);

return (
    <MDX
        components={{
            a: SafeLink,
        }}
    />
);
```
