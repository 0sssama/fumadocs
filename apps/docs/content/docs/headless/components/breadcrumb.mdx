---

A hook for implementing Breadcrumb in your documentation, it returns the path to
a page based on the given page tree.

## Usage

it exports a `useBreadcrumb` hook:

```tsx
import { useBreadcrumb } from 'next-docs-zeta/breadcrumb'

const tree: TreeNode[] = buildPageTree(allMeta, allDocs)
const pathname = usePathname()

const items = useBreadcrumb(pathname, tree)
```

### Breadcrumb Item

| Prop   | Type             | Description  |
| ------ | ---------------- | ------------ |
| `name` | `string`         | Name of item |
| `url`  | `string \| null` | Href link    |
