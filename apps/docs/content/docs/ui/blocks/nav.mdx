The default Navbar component with a sidebar toggle and search dialog trigger.

<Callout title="Note">It has been included in the layout by default.</Callout>

## Usage

```tsx
import { Nav as OriginalNav } from 'fumadocs-ui/nav';

export function Nav() {
  return <OriginalNav title="My App" enableSidebar collapsibleSidebar />;
}
```

### Disable Sidebar

As your navbar can be displayed in any pages without a sidebar, you may want to
hide the sidebar trigger in specific pages.

```tsx
<Nav enableSidebar={pathname.startsWith('/docs/') || pathname === '/docs'}>
  ...
</Nav>
```

### Items

You can pass an array of items to the component.

```tsx
const items = [
  {
    text: 'Documentation',
    url: '/docs',
    external: false,
  },
];
```

### Links

You can pass an array of links to the component, it needs an icon and label (for
`aria-label`).

```tsx
const links = [
  {
    label: 'Github',
    icon: <GithubIcon className="size-5" />,
    href: 'https://github.com/fuma-nama/next-docs',
    external: true,
  },
];
```
