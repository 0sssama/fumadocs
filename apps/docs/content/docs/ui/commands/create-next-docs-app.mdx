## Usage

No arguments needed.

<Tabs items={["npm", "pnpm", "yarn"]} defaultValue="npm">
<Tab value="npm">

```bash
npm create next-docs-app
```

</Tab>
<Tab value="pnpm">

```bash
pnpm create next-docs-app
```

</Tab>
<Tab value="yarn">

```bash
yarn create next-docs-app
```

</Tab>
</Tabs>

### Options

You can choose one of two examples: `simple` and `advanced`.

- Simple: with Typescript and Next.js
- Advanced: with Tailwind CSS, Typescript and Next.js
