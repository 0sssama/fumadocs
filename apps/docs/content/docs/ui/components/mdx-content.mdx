## Usage

Wrap your content in the component.

```tsx
import { MDXContent } from 'next-docs-ui/mdx';

function Component() {
  return <MDXContent>...</MDXContent>;
}
```

It uses the Tailwind CSS Typography plugin internally, you may customise it with CSS.

### With Tailwind CSS

Our Tailwind CSS plugin overrides the default configuration for Typography plugin. You can use the `prose` utility directly:

```tsx
<div className="prose">...</div>
```
