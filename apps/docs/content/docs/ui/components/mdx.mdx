## Usage

The default components include [MDX Content](/docs/ui/components/mdx-content), [Card](/docs/ui/components/card), [Code Block](/docs/ui/components/codeblock) and [Heading](/docs/ui/components/heading).

For Client Components:

```ts
import defaultComponents from 'next-docs-ui/mdx';
```

For Server Components:

```ts
import defaultComponents from 'next-docs-ui/mdx-server';
```
