## Usage

The default components include [Card](/docs/ui/components/card), [Code Block](/docs/ui/components/codeblock) and [Heading](/docs/ui/components/heading).

For Client Components:

```ts
import defaultComponents from '@fuma-docs/ui/mdx/default-client';
```

For Server Components:

```ts
import defaultComponents from '@fuma-docs/ui/mdx/default';
```
