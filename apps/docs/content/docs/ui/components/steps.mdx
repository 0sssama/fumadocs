## Usage

Put your steps into the `Steps` container.

```mdx
import { Step, Steps } from 'next-docs-ui/components/steps'

<Steps>
<Step>

### Hello World

</Step>

<Step>

### Hello World

</Step>
</Steps>
```

### Without imports

You can directly use the CSS classes without importing it.

```mdx
<div className="nd-steps">
  <div className="nd-step" />
</div>
```
