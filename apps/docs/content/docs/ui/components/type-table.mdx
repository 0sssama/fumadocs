> it is an optional component which won't be loaded by default.

## Usage

It accepts a `type` property.

```mdx
import { TypeTable } from '@fuma-docs/ui/components/type-table';

<TypeTable
  type={{
    percentage: {
      description:
        'The percentage of scroll position to display the roll button',
      type: 'number',
      default: 0.2,
    },
  }}
/>
```

### Type Table

<TypeTable
  type={{
    type: {
      description: 'The type of object',
      type: 'Record<string, ObjectType>',
      typeDescriptionLink: '#object-type',
